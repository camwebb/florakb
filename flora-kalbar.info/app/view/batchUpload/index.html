Langkah-langkah dalam Batch Upload
  1. Jalankan aplikasi FTP seperti FileZilla
  2. Login dengan Username dan Password yang sama dengan Anda login ke dalam situs flora-kalbar.info
  3. Upload file zip Anda ke dalam folder ZIP_FILES_HERE
  4. Jika langkah 1-3 telah dilakukan, klik tombol Lanjut.
